# Development notes

## Vue 2 and Vue DevTools

This project uses **Vue 2**. Vue DevTools **v7** only supports Vue 3 and will not work here.

- Use the **legacy** Vue DevTools extension that supports both Vue 2 and Vue 3.
- In Chrome: search the Web Store for “Vue.js devtools” and install the **legacy** version (or the one that explicitly supports Vue 2).
- In Firefox: install the **legacy** Firefox extension from the Vue DevTools releases/legacy page.
- If you still use v5, that also works with Vue 2.

**Rule of thumb:** Install and enable **only** the legacy (Vue 2–compatible) version when working on this project; disable or uninstall v7 to avoid the “Vue DevTools v7 detected in your Vue2 project” message.
